⚡ Overview

ReconX is a lightweight yet powerful OSINT and reconnaissance toolkit developed in Python. It provides multiple information-gathering modules that help users collect publicly available information about domains and network infrastructure from a single terminal interface.

Whether you're learning cybersecurity, practicing OSINT techniques, or solving CTF challenges, ReconX offers a simple and efficient way to perform basic reconnaissance tasks.

«⚠️ This project is intended strictly for Educational, Research, and CTF Purposes Only.»

---

✨ Features

🌐 IP Address Lookup

Resolve a domain name into its corresponding IP address.

📍 IP Geolocation

Gather information such as:

- Country
- Region
- City
- ISP
- Organization
- Timezone
- Coordinates

📋 WHOIS Lookup

Retrieve domain registration details including:

- Registrar
- Creation Date
- Expiration Date
- Updated Date
- Name Servers

🔎 DNS Records Enumeration

Discover common DNS records:

- A Records
- MX Records
- NS Records
- TXT Records
- CNAME Records

📡 HTTP Header Analysis

Inspect server response headers to gather useful web server information.

🔍 Subdomain Discovery

Check for common subdomains including:

- www
- mail
- ftp
- admin
- api
- blog
- dev
- test
- portal

💥 Full Recon Scan

Run all reconnaissance modules together and generate a complete overview of the target domain.

📦 Installation

Clone the repository:

git clone https://github.com/yourusername/ReconX.git
cd ReconX

Install dependencies:

pip install requests colorama python-whois dnspython

Run ReconX:

python reconx.py
